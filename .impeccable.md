## Design Context

### Users
Personal de administración interactuando con alta frecuencia. Su contexto es de trabajo rápido, gestionando pedidos por teléfono o WhatsApp al mismo tiempo. Necesitan máxima claridad, legibilidad y botones de acción rápidos (Job: cargar ventas, revisar y mover stock de garrafas pesadas sin errores).

### Brand Personality
Confiable, Robusta, Industrial, Eficiente. El tono es directo y profesional. "Patrón del Gas" inspira liderazgo en la distribución.

### Aesthetic Direction
**Industrial-Minimalist Admin (Blue Theme)**. 
- Paleta de colores restringida: Azul marino profundo (extraído del logo) como color principal, fondos limpios, acentos técnicos para estados.
- Cero "glassmorphism" o sombras genéricas. Bordes definidos, áreas de contenido estructuradas.
- Tipografía moderna pero utilitaria (estilo Inter/Roboto Mono para numéricos).
- Anti-referencia: Nada de bordes muy redondeados (border-radius grandes), colores pastel o animaciones elásticas. 

### Design Principles
1. **Densidad con Claridad**: Los datos importan más que la decoración. Usar ritmo espacial para separar bloques lógicos sin usar "tarjetas dentro de tarjetas".
2. **Jerarquía Funcional**: Las acciones principales (Nueva Venta, Registrar Stock) deben dominar visualmente con colores de acento vibrantes.
3. **Escaneabilidad Numérica**: El dinero y el stock deben leerse al instante. Las deudas (préstamos) deben generar tensión visual (rojo fuerte) para no ser ignoradas.
4. **Respuesta Táctil Rápida**: Todo elemento interactivo debe cambiar de estado de forma reactiva (snappy) y sin demoras.
